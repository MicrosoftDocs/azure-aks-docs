> [!IMPORTANT]
> Starting on **September 30, 2027**, Azure Kubernetes Service (AKS) no longer supports the Open Service Mesh (OSM) add-on. The upstream OSM project has also been retired by the [Cloud Native Computing Foundation (CNCF)](https://docs.openservicemesh.io/). [Migrate any existing OSM configurations to equivalent Istio configurations](/azure/aks/open-service-mesh-istio-migration-guidance). For more information on this retirement, see [Retirement: Open Service Mesh (OSM) add-on for AKS](https://azure.microsoft.com/updates?id=open-service-mesh-add-on-for-aks-will-be-retired-on-september-30-2027).
