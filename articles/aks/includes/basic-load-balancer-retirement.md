> [!IMPORTANT]
> Starting on **September 30, 2025**, Azure Kubernetes Service (AKS) no longer supports Basic Load Balancer. We recommend using Standard Load Balancer for new deployments and [upgrading any existing deployments to the Standard Load Balancer](azure/load-balancer/load-balancer-basic-upgrade-guidance). For more information on this retirement, see [Basic Load Balancer retirement update](https://azure.microsoft.com/updates/azure-basic-load-balancer-will-be-retired-on-30-september-2025-upgrade-to-standard-load-balancer/).
